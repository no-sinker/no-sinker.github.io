フラメンコのレッスン

26 5月
Tweet

子供フラメンコレッスン、水曜日に猪子家でやってます。まだ一年経ってませんが、セビジャーナスの4番まで組んで踊れます。

セビ3番！？

藍ちゃんは、うちの志帆とおなじく年長で、未来ちゃんは小一です。

細かいところはまだ間違ったりしますが、だんだんとできるようになってきています。組んで踊ると楽しいみたいで、キャッキャ言いながら楽しんでます。難しい部分ができたねえ！なんて言うとペアで、ハイタッチしたりして、喜んでます。楽しいまま収集つかなくなることもあるけど・・・・（苦笑）。

 

ミララカラ（セビジャーナス）って昔、帝塚山スタジオ時代、先生が歌ってくれてたなあと思い返して、ネットで歌詞を調べて、で歌い方を復習し、皆で少し練習してみました。

志帆は踊りよりもセビの歌の方が楽しいようです。一緒に練習したのがよほど楽しかったのか、お風呂の中でも歌い、歌いながら、踊ってみて、あれ？ここはどうなるんや？と一人でぶつくさ・・・。そうそう、そうやっていくと上達するよ・・・。ほんとに歌いながら踊れるようになれたらいいなと思います。

 

めっちゃかっこいいーー！！

あと、今はセビジャーナスのパルマと足（アクセント）もやっていて、シューズを履いての練習も計画中です。

左から未来ちゃん、藍ちゃん、志帆です。

レッスンが終わってから、youtubeで、子供がアレグリアスを踊っているのを見ていたら、藍ちゃんが真剣に見ていました。そして、藍ちゃんママが、藍はいつも家で手叩いてると言っていました。よほど好きなんだね！

皆に負けないように、指導できるよう、私もがんばります！！！

めちゃ真剣です

大人のクラスは自治会館（よつば会館になりました）にて。木曜の9時～10時半の部はこんな感じです。

アレグリアスを練習中です

4月から、少しずつまたメンバーが増え、チャコットで今も習われている素敵な方が入ってきて下さいました！

パルマの叩き方から一緒に、練習しています。みんなすこしずつ新しいことにチャレンジ！それぞれの良い所を感じていますが、うまく伸ばしていけたらと思っています。

木曜の10時半～１２時の部は、今日からカスタネットを始めました。セビをほぼ1年やって、カスタネットを鳴らしてみると、結構みなさん、楽しそう！ちょうど、今が練習を始めるのにいい頃です。

みんなに刺激され、めっちゃがんばらなあかん！！と危機感感じています（笑）。

今年も薔薇が咲きました

24 5月
Tweet

バラアーチ

地面にえっさほいさと穴を掘って、アーチにコンクリートの足を履かせて、埋めて、肥料やらなんやら色々混ぜた土と薔薇を2種類植えてかれこれ3年。

こんなにアーチに絡んでくれるとは！！かわいいです。虫もつくし、病気にはかかりやすいし、手間がかかるけれど、きれいな葉っぱが生い茂ってきて、（秋～冬の間は葉が落ちるんです。）つぼみがたくさんになってくると、ワクワクドキドキ！！

ピエールドロンサール

やっぱり薔薇はいいですね。。。。多分１週間くらいが見ごろだと思いますが・・・

黄色はシャルロットという品種

もうすぐ、ブラックベリーの季節。つるはにょきにょき、はっぱはたくさん生い茂ってます。お花はまだだけど、白くてかわいいんですよ。虫もつかないし、全然手間がかからず、フェンスをかわいくしてくれる優等生！

今年もプチトマトの大量収穫を狙い、何種類か植えてみました。

トスカーナバイオレット（紫色のプチトマト）。まだお花だけ

プチトマトは1株植えるだけで、ほんとに、１シーズン全く買わなくてもいい位に、毎日２～30個採れるようになるんです。美味しいし、かわいい！赤いのと黄色いのも植えました。どれが、おいしいかまた報告します。大き目の鉢やプランターでベランダでもできるので、ぜひお勧めです。

今年から仲間入りは、紫色のプチトマト！ぶどうのようにぷるんと柔らかい実がなるそう。。すっごくたのしみ！！

あと、ナス、しそ、パプリカ、バジル、ゴーヤ、きゅうりを植えました。ゴーヤときゅうりは、緑のカーテンになるように、窓に植えてます。２年前にもゴーヤを植え、素敵な日よけになってくれて、私は大喜びでしたが、娘は、ゴーヤを微妙な顔で見ていました。（笑）苦いから、たくさん消費できないんですよね（笑）。

今年も手間がかからなくて、いっぱい採れるの、狙ってマス（笑）。

かーべさんにドレス作ってもらっています

16 5月
Tweet

型紙のためのドレス

ウエディングドレスとフラメンコドレスのデザイナーかーべさんにドレスをお願いしました。うちのおちびの頭がソフトクリームみたいですね（笑）。ちなみにブローはしてませんよ（笑）。

かーべさんはお仕事がとってもとても丁寧！センスもよくて、こうしたい！という投げかけにかーべさんのアイディアをプラスして、形にしてくれます。

前回はデザインの相談、採寸、生地選びまでやったので、今日、事務所に行くとすでにトワールができあがっていました。嬉しい！この段階でめっちゃきれい！！

トワールは、実際の生地で縫う前に、それ用の生地で縫って詰める所や、直すところをメモしたり、マークしたりするだけの型紙を作るためのドレスとの事なのですが、フラメンコドレスを作るためにその、もう一着分の手間を惜しまないところもすごいです。。。私は、かーべさん作の輝先生のドレスやフタキちゃんのドレスを見てきたから良く分かるのですが、縫い目一つ一つがめちゃくちゃきれいなんです。すっごく丁寧で、繊細、仕事に対する愛情がにじみ出ています。

今の時代でその丁寧なお仕事ぶりは惚れます。素敵！次回が楽しみ！！

スペイン行きたい

09 5月
Tweet

歌うますぎ少年

もう、ほんまにすぐにでもスペイン行きたい。フラメンコが好きすぎて、恋してる人になかなか会えないみたいな苦しい気持ち。行ったら帰って来れなくなったりして。。。以前はそれほどでもなかったんだけど、いよいよほんまに行きたくてたまらなくなってきました。。。

今はyoutubeという便利なものがあるので、好みのflamencoがたくさん見れて、かなりラッキー。古いものから新鮮なものまで。

友人のフラメンコ仲間フタキちゃんが今スペインに行っていて、以前行った時も相当良かったらしいけれど、今回メールしてみたら、フェルナンド・モレナという人がいいと聞いたので、早速色々調べたら、やっぱり好みのはブレリアだ！

めっちゃいい！！こんなんを実際に聴いたら、観たらやばい！！いいなあフタキちゃん！

フェルナンドモレナ

フラメンコって・・・体ひとつでこんなにすばらしいってすごい！ブレリア漬けになりたい！スペイン人がおばあちゃんになっても、おじいちゃんになってもブレリアで周りの人から心からole！って言わせられるのが素敵過ぎる！

年齢いってても、フラメンコを心から楽しめるスペインの人の感受性が若いなあって思います。まだyoutubeでしか知らないけれど・・・

歌い手も踊り手もギターの人も、その人次第で生涯現役という感じ。しかし、こないだ、テレビでスペインの失業率が高すぎと指摘されていたけど、フラメンコというお楽しみがあるから、物欲があまりないからでしょうか。。。国民から、文化が愛されているのはすごくいい事ですけどね。そして、そういう文化があるのがうらやましい。

新作アクセサリー2種

05 5月
Tweet

いよいよゴールデンウィークも終わりにさしかかってきましたが、いかがお過ごしでしょうか？

今日なんかは、外を歩くと暑いくらいでした。肌寒い日々から一気に初夏になってきましたね。

個人的にはたくさんワンピースが着られる嬉しい季節！の到来がウキウキです。
