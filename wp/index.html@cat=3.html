フラメンコのレッスン

26 5月
Tweet

子供フラメンコレッスン、水曜日に猪子家でやってます。まだ一年経ってませんが、セビジャーナスの4番まで組んで踊れます。

セビ3番！？

藍ちゃんは、うちの志帆とおなじく年長で、未来ちゃんは小一です。

細かいところはまだ間違ったりしますが、だんだんとできるようになってきています。組んで踊ると楽しいみたいで、キャッキャ言いながら楽しんでます。難しい部分ができたねえ！なんて言うとペアで、ハイタッチしたりして、喜んでます。楽しいまま収集つかなくなることもあるけど・・・・（苦笑）。

 

ミララカラ（セビジャーナス）って昔、帝塚山スタジオ時代、先生が歌ってくれてたなあと思い返して、ネットで歌詞を調べて、で歌い方を復習し、皆で少し練習してみました。

志帆は踊りよりもセビの歌の方が楽しいようです。一緒に練習したのがよほど楽しかったのか、お風呂の中でも歌い、歌いながら、踊ってみて、あれ？ここはどうなるんや？と一人でぶつくさ・・・。そうそう、そうやっていくと上達するよ・・・。ほんとに歌いながら踊れるようになれたらいいなと思います。

 

めっちゃかっこいいーー！！

あと、今はセビジャーナスのパルマと足（アクセント）もやっていて、シューズを履いての練習も計画中です。

左から未来ちゃん、藍ちゃん、志帆です。

レッスンが終わってから、youtubeで、子供がアレグリアスを踊っているのを見ていたら、藍ちゃんが真剣に見ていました。そして、藍ちゃんママが、藍はいつも家で手叩いてると言っていました。よほど好きなんだね！

皆に負けないように、指導できるよう、私もがんばります！！！

めちゃ真剣です

大人のクラスは自治会館（よつば会館になりました）にて。木曜の9時～10時半の部はこんな感じです。

アレグリアスを練習中です

4月から、少しずつまたメンバーが増え、チャコットで今も習われている素敵な方が入ってきて下さいました！

パルマの叩き方から一緒に、練習しています。みんなすこしずつ新しいことにチャレンジ！それぞれの良い所を感じていますが、うまく伸ばしていけたらと思っています。

木曜の10時半～１２時の部は、今日からカスタネットを始めました。セビをほぼ1年やって、カスタネットを鳴らしてみると、結構みなさん、楽しそう！ちょうど、今が練習を始めるのにいい頃です。

みんなに刺激され、めっちゃがんばらなあかん！！と危機感感じています（笑）。

かーべさんにドレス作ってもらっています

16 5月
Tweet

型紙のためのドレス

ウエディングドレスとフラメンコドレスのデザイナーかーべさんにドレスをお願いしました。うちのおちびの頭がソフトクリームみたいですね（笑）。ちなみにブローはしてませんよ（笑）。

かーべさんはお仕事がとってもとても丁寧！センスもよくて、こうしたい！という投げかけにかーべさんのアイディアをプラスして、形にしてくれます。

前回はデザインの相談、採寸、生地選びまでやったので、今日、事務所に行くとすでにトワールができあがっていました。嬉しい！この段階でめっちゃきれい！！

トワールは、実際の生地で縫う前に、それ用の生地で縫って詰める所や、直すところをメモしたり、マークしたりするだけの型紙を作るためのドレスとの事なのですが、フラメンコドレスを作るためにその、もう一着分の手間を惜しまないところもすごいです。。。私は、かーべさん作の輝先生のドレスやフタキちゃんのドレスを見てきたから良く分かるのですが、縫い目一つ一つがめちゃくちゃきれいなんです。すっごく丁寧で、繊細、仕事に対する愛情がにじみ出ています。

今の時代でその丁寧なお仕事ぶりは惚れます。素敵！次回が楽しみ！！

男の子無事出産しました

17 4月
Tweet

相当長い間更新してなくてすみません・・・（笑）

1月11日に2430gの元気な男の子を出産いたしました。出産、そして大震災があって短い間に生死を体感したこのごろでした。

あの大震災で、家族や友人が元気に生活できていること、日々の当たり前だと思っていた生活がどれだけありがたいものだったかを考えさせられました。人間はいつも死と隣り合わせ。

だからこそ、会いたいと思った人にはすぐに会おう、周りの人を大切にしよう、毎日の生活を一日一日を大切に生きて行こう、ボランティアや寄付ももちろん大切ですが、被災者のためにも、一人ひとりが出来ることを意識して生産性を上げていって、まず日本経済を豊かにする事が、結果被災者を助けることになると思っています。暗くなってはいけない。元気のある私達がどんな分野でもいい、世界から注目されるくらいに輝くものを発信していかなくては、引っ張っていかなくてはいけないと思います。被災者の方を勇気付けられるのは、言葉ではないと思います。元気がある私達のこれからのがんばりにかかっていると思います。今世界から見られている日本のマイナスイメージを吹き飛ばせる技術力や仕事の丁寧さはどこの国にも負けないはずです。そのために自分が出来ることは小さな事だけれども、本当に心を込めて仕事をやっていこうと胸に誓いました。

アクセサリーとフラメンコは人を笑顔にしてくれるものだと思います。

フラメンコと妊婦生活

03 11月
Tweet

只今私は、妊娠7カ月です。フラメンコ、元気にやっています。
出血や前置胎盤もなく、逆子になったこともなく、体重増加も毎日チェックして順調です。

おなかの赤ちゃんも元気に動き回っています。
大体一カ月に一キロずつの増加で、最終的に10キロくらいのプラスが好ましいそうです。
とはいえ、体重をコントロールするのは大変で、こないだ病院に行ったら待合室で一緒だった妊婦さんが、頑張ったのに2キロ増えてたと嘆いていました。めっちゃわかるわ～と思いながら聞いてました。

妊娠初期を思い返してみると、8月の中ごろまで、つわりのため、胃が常にムカムカして、ごはんが美味しく食べられず、だるさもあり、一日中横になったりしていました。丁度、地元けやき坂の納涼祭のあたりが一番気持ち悪かったんです。
中期に入ると、つわりがやっと収まって来て、胃がおさえつけられていたのが急にふっとなくなり、逆にいくら食べても満腹感があまりなく、1ヶ月の間に、あっという間に2.5キロも増え、母子手帖に体重計コントロールの判子を押されてしまいました(^_^;)。
久しぶりにご飯が美味しく食べれるようになると反動で食べちゃうんですよね。最初の方はやっぱり、激しいフラメンコは不安だったので、お休みしていましたが、来年の桜祭りに新しいたのしい演目を踊りたいと思い、新しい演目を教えてもらえるよう、輝先生にお願いしました。先生は体調を気遣ってくださり、ゆっくり、すこしずつ無理ない感じでレッスンしてくれています。

丁度フラメンコ復活して、一ヶ月経ったところです。新しい演目は、カラコレスという演目です。色々と調べると、扇やバタデコーラ(フラメンコでは、バタデコーラという後ろの長ーいドレスを使うんです。足で綺麗に蹴って踊るんですよ。)を使ってよく踊られますが、そのまま体一つでも踊られているようですね。
教えてもらっているのが、マヌエルベタンソスの振付の、とっても曲線的な美しい踊りで、楽しくて仕方ありません(*^^*)！
最初は、カラコレスのイメージから、勝手にグワヒーラっぽい歌かと思っていたら、アレグリやカンティーニャに似たメロディーですね！まだまだ知らない曲だらけなので、日々勉強です。長調ですが、所々の歌の最後など短調の悲しげなメロディーになったり、また長調にもどったりするのが特徴かなと思います。

また、体調の話に戻りますが、このカラコレスを練習し出したあたりから、以前の体調が戻ってきました。体重の増加も順調ですし、便秘知らずで、いたって健康です！！
多分土日以外のほぼ毎日練習してるからですかね(*^^*)！練習のスタンスはほとんど妊娠前と同じような感じです。でも、そんなに激しい感じじゃないので、ゆったり練習してます。
サパテアード(ステップ)の練習も普通にやっていますが、いたって元気です。
何気ないアバニコ(扇)使いの練習だけでも研究の価値あります！たくさん出てくるマルカールのバリエーションもいい練習になります！一人でアバニコなしバージョン、左右対照と、ありバージョンで練習してます。きれいにアバニコを使いこなせるようになりたいです！

子供フラメンコクラス始めました！

22 9月
Tweet

先週から子供フラメンコクラス、開講しました！

大人のクラスは今年の3月からスタートしたのですが、子供が相手だと、どんな感じかなあと不安もありながら、やってみた、先週の水曜日。

お友達のお子ちゃん二人、最初の大人でも難しいセビジャーナスのステップから入ったのですが、すごい一生懸命集中してやってくれて、繰り返し練習するうち、できるようになり、その一生懸命さが本当に嬉しかったです。集中力が偉い！私も見習わないといけません・・・。

初めての子供レッスンの感想・・・スタンスは大人と一緒ということに気づきました。大人に対してとはまた違う反応の面白さはありますが（笑）

幼稚園に行くようになると、女の子は特に大人っぽくなりますね。大人に対してと同じ対等な感じで、十分大丈夫だなと思いました。逆に子供扱いしてはいけないなと思いました。

終始真剣そのものの表情でしたが、レッスン終了後、どうだった？と聞くと楽しかった！と声をそろえて言ってくれました。

そして、今週から、メンバーがもう一人増え、3人になりました！みんなけやきっ子です。

けやきっ子でも、そうでなくても大歓迎！

水曜日、午後２時～１時間、猪子家にてやってます。お月謝は月4回、２０００円です。

興味ある方はこちらまで。見学、体験無料です。
