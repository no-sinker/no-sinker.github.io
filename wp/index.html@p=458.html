手作りパン教室　hanapan

29 10月
Tweet

パン作りを習いに行ってきました！

親子でフラメンコに来てくれている、舟川さんと、お子さんが習いに来てくれている矢野さんと私の三人で、けやき坂3丁目にあります、パン作りの先生のご自宅、木村さん宅にお邪魔しました。

ポテマヨパン

カプチーノロイヤル

今日教えていただいたのは、ポテマヨパン、カプチーノロイヤルの二品です。

10時から作業開始、あっという間にお昼になり、木村さんお手製のサラダや、ライ麦パン（天然酵母で、あらかじめ用意してくださっていて、ちょうどお昼にほかほかの焼きたて！！）試食のポテマヨパン、コーヒーに、お手製のかぼちゃプリンまで用意して下さり、美味しく頂きました！本当にどれもめちゃくちゃ美味しくて、癒されました。。。感動感激いたしました。。。ご馳走様でした！！！

見た目もとても綺麗！！

手づくりパン教室の名前、　hanapan　の由来は木村さんが飼っていらっしゃる、ミニチュアダックスフンドのhanaちゃんから付けたそうです。とってもかわいいおりこうなわんちゃんにも癒されました。

一口食べると材料の良さがしみじみ分かる、木村さんの手作りパンは、粉にもこだわっていて、すべて国産小麦だそうです。輸入の粉は、海を渡る際に、生じる湿気のカビ対策の大量の薬が、かなり体に悪いそうなんです。色々これからも教えて下さいね！

11月のメニューは

A　　じゃがじゃがパイ　と　いちじく＆くるみブレッド

B　　じゃがじゃがパイ　と　お抹茶＆あずきロール

１２月のメニューは

パネトーネとシュトーレン

です。

１２月は人気ですでに予約が埋まってきているそうです。お早めにどうぞ！

ご予約の方は、木村さんにお電話（072-798-0212 ・ 090-7364-7754）か直接予約してください。

レッスン受けられるのは一度に3人までとなっています。

参加費用は　2000円　　（初回のみ1500円）　です。

住所は、けやき坂3丁目10－10　木村　浩子さん宅にて
